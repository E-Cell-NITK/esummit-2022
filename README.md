# Repository for E-Cell NITK's E-Summit 2022 React App

External packages installed
- chakraUI & its dependencies
- react-router-dom & its dependencies
- react-icons

## Running the app

- `npm run start`
- check [localhost:3000](http:localhost:3000) to view the app

## Building the app

- `npm run build`
